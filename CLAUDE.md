# DIZZY - Event-Driven Architecture Framework

DIZZY is a software philosophy for building event-driven systems with decoupled infrastructure.
Core components: Commands (c), Events (e), Procedures (d), Policies (y), Projections (j), Models (m), Queries (Q).
